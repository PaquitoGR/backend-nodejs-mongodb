# NodeApp

Webside and API application.

## Install

Install dependencies:

```sh
$ npm install
```

Review database connection on /lib/connectMongoose.js (see "Start a MondgoDB server")

## Start

In production:
```sh
$ npm start
```

In development:
```sh
$ npm run dev
```
